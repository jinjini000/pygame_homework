## 🚀 미사일 회피 게임 (Missile Evasion Game)

이 프로젝트는 Pygame 라이브러리를 사용하여 구현된 간단한 미사일 회피 아케이드 게임입니다. 플레이어는 미사일이 끝없이 생성되는 화면에서 비행기를 조종하여 최대한 오래 살아남고 높은 점수를 얻어야 합니다.

-----

### ✨ 주요 기능

  * **플레이어 조종:** 방향키(화살표)를 사용하여 비행기를 조종합니다.
  * **물리 기반 움직임:** 가속도, 최고 속도, 감속(항력)이 적용된 현실적인 움직임을 제공합니다.
  * **적 미사일:** 화면 밖에서 미사일이 생성되어 플레이어를 추적합니다.
  * **경고 시스템:** 화면 밖에서 다가오는 미사일의 방향을 노란색 화살표로 알려줍니다.
  * **플레어(Flare) 회피 기술:** **Spacebar**를 눌러 플레어를 발사하여 미사일의 추적을 일시적으로 방해할 수 있습니다. (횟수 제한 있음)
  * **점수 시스템:** 생존 시간에 비례하여 점수가 증가하며, 점수가 높아질수록 미사일 생성 속도가 빨라집니다.
  * **게임 상태:** 카운트다운, 게임 플레이, 게임 오버 상태를 가집니다.

-----

### 🕹️ 게임 방법

#### **1. 목표**

미사일에 맞지 않고 **가장 오랫동안 생존**하여 최고 점수를 달성하세요.

#### **2. 조작법**

| 키(Key) | 동작 (Action) |
| :---: | :--- |
| **↑/↓/←/→** (방향키) | 플레이어(비행기) 이동 및 가속 |
| **Spacebar** (스페이스바) | **플레어(Flare) 발사** (미사일을 유인) |
| **R** (게임 오버 시) | 게임 재시작 |

#### **3. 플레어 사용**

  * 플레어는 미사일의 추적 대상을 일시적으로 플레어의 위치로 변경시켜 플레이어에게 회피할 시간을 벌어줍니다.
  * 사용 가능 횟수는 화면 좌측 상단에 표시되며, 최대 3개로 제한됩니다.

-----

### 🛠️ 실행 환경 및 설치

이 프로젝트를 실행하기 위해서는 **Python**과 **Pygame** 라이브러리가 필요합니다.

#### **1. 필수 조건**

  * Python 3.x
  * Pygame

#### **2. Pygame 설치**

터미널 또는 명령 프롬프트에서 다음 명령어를 사용하여 Pygame을 설치할 수 있습니다.

```bash
pip install pygame
```

#### **3. 실행**

1.  코드 파일 (예: `missile_game.py`)과 `img` 폴더 내의 이미지 파일들 (`player_plane.png`, `missile.png`)이 같은 경로에 있는지 확인합니다.
2.  터미널에서 코드를 실행합니다.

<!-- end list -->

```bash
python missile_game.py
```

### 💻 코드 구조

프로그램은 클래스 기반으로 구성되어 있어 각 요소의 역할이 명확합니다.

| 클래스 (Class) | 설명 (Description) |
| :--- | :--- |
| `Colors` | 게임에 사용되는 색상 값(RGB 튜플)을 정의합니다. |
| `Fonts` | 게임 내 텍스트 출력에 사용되는 폰트 객체를 정의합니다. |
| `Consts` | 화면 크기, 속도, 가속도, 게임 상태 등 모든 게임 상수들을 관리합니다. |
| `MyDisplay` | Pygame 화면(Screen) 및 창 제목을 설정합니다. |
| `ImgLoad` | 게임에 필요한 이미지 파일들을 로드하고 크기를 조정합니다. |
| `Flare` | 미사일을 유인하는 **플레어 개체**의 상태(위치, 수명) 및 그리기 로직을 담당합니다. |
| `Missile` | **미사일 개체**의 상태(위치, 속도), 이동 및 추적 로직(플레어 유인 포함)을 담당합니다. |
| `Func` | 미사일 무작위 생성 위치 결정, 경고 표시, 텍스트 출력 등 유틸리티 함수들을 모아놓았습니다. |
| `Main` | 게임의 메인 루프, 이벤트 처리, 게임 상태 업데이트, 그리고 화면 렌더링을 총괄합니다. |

### 📄 상수 요약 (Consts Class)

| 상수 (Constant) | 값 (Value) | 설명 (Role) |
| :--- | :--- | :--- |
| $\text{screen\_width}$ | $1200$ | 게임 화면 너비 |
| $\text{screen\_height}$ | $720$ | 게임 화면 높이 |
| $\text{player\_max\_speed}$ | $7.0$ | 플레이어의 최대 이동 속도 |
| $\text{missile\_max\_speed}$ | $17.0$ | 미사일의 최대 추적 속도 |
| $\text{flare\_duration\_ms}$ | $1500$ | 플레어가 활성화되어 미사일을 유인하는 시간 (밀리초) |
| $\text{max\_flares}$ | $3$ | 플레이어가 소지할 수 있는 최대 플레어 갯수 |
| $\text{score\_spawn\_interval}$ | $1000$ | 이 점수 간격마다 미사일이 하나씩 추가 생성됩니다. |
| $\text{warning\_time\_ms}$ | $1500$ | 미사일 생성 전 경고가 표시되는 시간 (밀리초) |
